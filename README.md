# MILO - Match Information & Logging Organizer

MILO (Match Information & Logging Organizer) is a web-based tool designed to help organize, track, and log competitive matches in real-time.

---

## Features

* **Dual Timers**: Track time for two teams simultaneously, with optional time caps.
* **Round Recording**: Automatically record the outcome of each round based on timer completion and points.
* **Points & Warnings**: Easily manage points and warnings for each team, with quick increment/decrement controls.
* **Transcripts**: a clear summary of all rounds, including scores, warnings, and winners.
* **Saving**: Saves your timers, points, warnings, and transcript in the browser so you can resume where you left off.
* **Customizable Settings**:

  * Enable or disable time caps
  * Set cap duration (minutes & seconds)
  * Adjust start time for timers
* **Easy Copying**: Copy the match transcript to clipboard for reporting or sharing.
* **Responsive Design**: Works across desktop and mobile browsers.

---
## Customization

* **Time Cap**: Enable or disable in settings, and set minutes and seconds.
* **Start Time**: Adjust the initial timer value.
* **Points & Warnings**: Increment/decrement manually or via long-press for rapid increase.
